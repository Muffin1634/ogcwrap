◆ f32

float f32

32 bit IEEE-754 single-precision floating-point

This type is a IEEE-754 single-precision floating point number. It allocates the highest bit for the sign, the next 8 bits for the exponent, and the low 23 bits for the mantissa. As such, it has 24 bits of precision, because the highest bit of the mantissa will always be 1 in binary, so it is not explicitly stored.

64 bit IEEE-754 double-precision floating-point

This type is a IEEE-754 double-precision floating point number. It allocates the highest bit for the sign, the next 12 bits for the exponent, and the low 52 bits for the mantissa. As such, it has 53 bits of precision, because the highest bit of the mantissa will always be 1 in binary, so it is not explicitly stored.
