Detailed Description

Header file for the wrapper of the Accelerated (Extended?) SouND subsystem.

<aesnd/aesnd.hpp>

This header contains functions in the ogcwrap::aesnd namespace for initializing the AESND subsystem, managing and gathering statistics from the Digital Sound Processor (DSP), and managing individual voices within it.
