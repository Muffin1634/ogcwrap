◆ ASNDReturnValue

enum struct ogcwrap::asnd::ASNDReturnValue : s8
strong

Enum to specify common return values of ASND functions.

Enumerator
OK 

This return value is used to indicate that a function was successful.

Invalid 

This return value is used to indicate that an input to a function is invalid.

NotAVoice 

This return value is used to indicate that the voice number given is invalid.

VoiceBusy 

This return value is used to indicate that a voice is busy.

VoiceUnused 

This return value is used to indicate that a voice is currently unused and is ready to be used.

VoiceWorking 

This return value is used to indicate that a voice is working on playing samples.

VoiceWaiting 

This return value is used to indicate that a voice is waiting for (more samples?).

◆ ASNDVoiceFormat

enum struct ogcwrap::asnd::ASNDVoiceFormat : u8
strong

Enum to specify the format of PCM samples that the DSP will use.

Enumerator
MonoU8 

Specifies that the PCM samples only have one sound channel (mono) and that they have unsigned 8-bit values.

MonoS8 

Specifies that the PCM samples are in mono and that they have signed 8-bit values.

MonoBE16 

Specifies that the PCM samples are in mono and that they have unsigned big-endian 16-bit values.

MonoLE16 

Specifies that the PCM samples are in mono and that they have unsigned little-endian 16-bit values.

StereoU8 

Specifies that the PCM samples have two sound channels (stereo) and that they have unsigned 8-bit values.

StereoS8 

Specifies that the PCM samples are in stereo and that they have signed 8-bit values.

StereoBE16 

Specifies that the PCM samples are in stereo and that they have unsigned big-endian 16-bit values.

StereoLE16 

Specifies that the PCM samples are in stereo and that they have unsigned little-endian 16-bit values.

bool ogcwrap::asnd::getSoundPlayStatus ( void  )

Checks whether sound is playing or not.

Note
true does not necessarily mean sound is playing; it just means that the sound is in play mode rather than pause mode. Voices may not have any samples to play and therefore may not be playing anything.
Returns
true if sound is playing, false otherwise
