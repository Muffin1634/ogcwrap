Detailed Description

Header file for the wrapper of the gu/matrix subsystem in libOGC.

<gu/gu.hpp>

This header contains functions in the ogcwrap::gu namespace for creating projection matrices, light projection matrices, and for manipulating the camera.

Warning
This wrapper of libOGC's gu subsystem does not include the math functions; use the related gxmath library for an update of those functions.
