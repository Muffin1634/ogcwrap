Initializes a perspective-style projection matrix for use with the GX subsystem.

Parameters
[out]matrix44The destination matrix that will be created; you must allocate this first.
[in]fovYThe field of view, in degrees, of the YZ plane (pitch).
[in]aspectThe aspect ratio of the resulting view.
[in]nearThe distance to the near clipping plane on the Z axis.
[in]farThe distance to the far clipping plane on the Z axis.
Returns
none; the result matrix is the first parameter.

void ogcwrap::gu::lookAt ( Mtx  camera34,
guVector *  position,
guVector *  upVector,
guVector *  target 
)

Resets the camera to be placed at position pointed at target.

The upVector parameter of the function determines the camera's roll. Normally this is (0, 1, 0) which refers to a normal upright camera. However, if you want your camera to tilt, then you can change the upVector.

Note
verbatim from libOGC: "The camera's reference viewing direction is the -z axis. The camera's reference 'up' direction is the +y axis."
Parameters
[out]camera34The camera matrix to modify.
[in]positionThe camera position in world space where the camera will be.
[in]upVectorThe "up" direction of the camera; this vector affects the camera's roll.
[in]targetThe target position in world space where the camera will be pointing.
Returns
none; the result matrix is in the first parameter.
