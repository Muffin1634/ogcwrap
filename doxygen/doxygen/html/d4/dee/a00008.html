Detailed Description

Header file for the wrapper of the Accelerated SouND subsystem.

<asnd/asnd.hpp>

This header contains functions in the ogcwrap::asnd namespace for initializing the ASND subsystem, managing and gathering statistics from tho Digital Sound Processor (DSP), managing individual voices, and testing sample buffers.
