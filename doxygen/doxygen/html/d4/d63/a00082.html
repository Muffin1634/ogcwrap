Detailed Description

Namespace containing functions related to the L2 cache.

Warning
All start_addr, src_addr and dst_addr arguments in this namespace MUST be aligned on a 32-byte boundary, and all length arguments in this namespace MUST be a multiple of 32, unless otherwise specified.
