◆ TVMode

enum struct ogcwrap::video::TVMode
strong

Enum to specify which video standard the Wii should use.

Enumerator
NTSC 

Use the NTSC standard found in North America and Japan.

PAL 

Use the PAL standard found in European countries.

MPAL 

Use the MPAL standard found in Brazil.

It is somewhat similar to the NTSC standard.

EuRGB60 

Use the EuRGB60 standard found in European countries.

Using 480 lines and runnning at 60Hz, this format has the same aspect ratio and timing as the NTSC standard.

DEBUG_NTSC 

Use the DEBUG_NTSC standard, for use with North America and Japan.

Note
A special decoder is needed to use this standard.
DEBUG_PAL 

Use the DEBUG_PAL standard, for use with European countries.

Note
A special decoder is needed to use this standard.
